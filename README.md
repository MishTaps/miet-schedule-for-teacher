# Просмотр расписания преподавателей в МИЭТ (Московском Институте Электронной Техники)

Это **не официальный** сайт администрации университета, а просто небольшой проект для студентов и преподавателей для расширения функционала просмотра расписания.

## Запуск программы локально

### Подготовка

1. Проверить, что установлен `Node.js`:

   ```powershell
   node -v
   npm -v
   ```

   При необходимости установить `Node.js`: <https://nodejs.org/en/download>

2. Скачать/клонировать репозиторий
3. Открыть консоль в корневой папке проекта
4. Ввести команду:

   ```powershell
   npm i
   ```

### Запуск в `build` режиме (для обычного просмотра страницы)

1. Ввести команду:

   ```powershell
   npm run build
   ```

2. Ввести команду:

   ```powershell
   npm run preview
   ```

3. В браузере открыть <http://localhost:4173/>

### Запуск в `dev` режиме (для разработчиков)

1. Ввести команду:

   ```powershell
   npm run dev
   ```

2. В браузере открыть <http://localhost:5173/>
